# Spreadshot
A UE5 exploratory prototype focused on weapons with the absurd power level of boomer shooters.

I don't have anywhere near a full picture of all the moving pieces in that template, so for now I decided to start a fresh FPS template and try to replicate that code as much as possible, according to our needs. 

Included is an html file containing my understanding of how the weapons work in the default UE5 Arena Shooter template, which can be opened in your web browser. This may get updated as I discover more so look carefully. (I like Zim, but I have to export in HTML/LaTEX/Markdown or else the .txts, or god forbid, .docx, are hard to read. In hindsight I probably could have used Markdown to save us all some time considering you're literally reading one right now).

This repo may not last long. I'm doing a little exploration with the code right now just to see how hard it'll be to remake some of this from scratch. If I hit a massive roadblock, we'll be moving back over to the Arena Shooter template and trying to figure it out from there, but my fingers are crossed that we can just do it here even if it's not pretty.
